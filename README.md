# DSA
Learning and solving DSA problems from LeetCode, HackerRank and GFG

1. Arrays
Basics: Traversal, Insertion, Deletion
Problems: Two Sum, Move Zeroes, Maximum Subarray
Advanced: Prefix sum, Sliding window, Kadane’s algorithm

2. Strings
Basics: Reversal, Palindrome check, Anagram check
Problems: Valid Palindrome, Longest Substring Without Repeating Characters
Advanced: String matching, KMP algorithm

3️. Linked Lists
Singly & Doubly linked lists
Operations: Insertion, Deletion, Traversal
Problems: Reverse Linked List, Detect cycle, Merge Two Sorted Lists

4️. Stacks
Basics: Push, Pop, Peek
Problems: Valid Parentheses, Min Stack
Advanced: Next Greater Element, Largest Rectangle in Histogram

5️. Queues
Basics: Enqueue, Dequeue
Problems: Implement Queue using Stacks, Circular Queue
Advanced: Sliding Window Maximum, Task Scheduler

6️. Hashing / Hash Tables
Basics: Hash maps, Frequency counting
Problems: Two Sum, Group Anagrams, Subarray Sum Equals K
Advanced: Hash sets for duplicates, Sliding window with hashmap

7️. Recursion & Backtracking
Basics: Factorial, Fibonacci
Problems: N-Queens, Word Search, Subsets
Advanced: Permutations, Combinations, Sudoku Solver

8️. Trees
Binary Trees: Traversals (Inorder, Preorder, Postorder)
Binary Search Trees (BST)
Problems: Maximum Depth, Lowest Common Ancestor, Validate BST
Advanced: Tree Diameter, Serialize/Deserialize Tree

9️. Heaps / Priority Queues
Basics: Min Heap, Max Heap
Problems: Kth Largest Element, Merge K Sorted Lists
Advanced: Heap Sort, Sliding Window Maximum

10. Graphs
Basics: BFS, DFS, Graph representation (adjacency list/matrix)
Problems: Number of Islands, Clone Graph
Advanced: Dijkstra, Bellman-Ford, Topological Sort, MST (Kruskal/Prim)

11. Dynamic Programming (DP)
Basics: Fibonacci, Climbing Stairs
Problems: Coin Change, Longest Common Subsequence, Maximum Subarray Sum
Advanced: DP on Trees, Knapsack problems, Matrix Chain Multiplication

1️2. Greedy Algorithms
Basics: Interval Scheduling, Activity Selection
Problems: Minimum Number of Arrows, Jump Game
Advanced: Huffman Coding, Fractional Knapsack

1️3. Bit Manipulation
Basics: AND, OR, XOR, Shift operations
Problems: Single Number, Counting Bits, Power of Two
Advanced: Subsets with bits, XOR tricks

1️4. Sliding Window & Two Pointers
Problems: Longest Substring with K Distinct Characters, Container With Most Water
Advanced: Maximum sum subarray of size K, Subarray Product Less Than K

1️5. Advanced Data Structures 
Trie: Auto-complete, Prefix search
Segment Tree / Fenwick Tree: Range sum queries
Disjoint Set / Union-Find: Connected components
